Favicon Dinamici per la Dashboard Meteo

Questa cartella contiene i diversi favicon che vengono utilizzati dalla dashboard meteo in base alle condizioni meteorologiche. Attualmente sono presenti favicon segnaposto che dovresti sostituire con icone reali.

Nota: I file .ico dovrebbero essere immagini vere, non file di testo. I file attualmente presenti sono solo segnaposto.

Elenco delle icone

Nome file Condizione meteo Quando viene usato
favicon-sunny.ico Cielo sereno, soleggiato Condizioni di default, cielo sereno
favicon-cloudy.ico Nuvoloso Quando il cielo è nuvoloso
favicon-rain.ico Pioggia Durante le precipitazioni
favicon-storm.ico Temporale Durante i temporali o con bassa pressione
favicon-foggy.ico Nebbia In condizioni di nebbia o foschia
favicon-snow.ico Neve Durante le nevicate
favicon-hot.ico Molto caldo Con temperature superiori a 30°C

Come sostituire le icone

Per sostituire queste icone con immagini reali:

  1. Crea o scarica le icone meteo desiderate
  2. Converti le immagini in formato .ico (16x16, 32x32 e/o 48x48 pixel)
  3. Sostituisci i file in questa cartella mantenendo gli stessi nomi
